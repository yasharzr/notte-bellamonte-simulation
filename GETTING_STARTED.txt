STEP 1: Install Node.js (one-time setup)
================================================

Download from: https://nodejs.org/
Choose LTS (Long-Term Support) version
Install with default settings


================================================
STEP 2: Install Dependencies (first time only)
================================================

Open Terminal/Command Prompt
Navigate to the notte-bellamonte-simulation folder
Run: npm install


================================================
STEP 3: Start the Server
================================================

In Terminal/Command Prompt, run:
  npm start

You should see:
  üç∑ Notte Bellamonte Simulation Server
  üìç http://localhost:3040


================================================
STEP 4: Open in Browser
================================================

FOR YOU (INSTRUCTOR):
  1. Open http://localhost:3040/instructor.html
  2. Click "Create Session"
  3. Copy the Session ID
  4. Share with students

FOR STUDENTS:
  1. Open http://localhost:3040/student.html
  2. Paste the Session ID
  3. Enter your name
  4. Click "Join Session"


================================================
FOR CLASSROOM (University WiFi)
================================================

BEFORE CLASS:
  1. Connect to WiFi
  2. Find your IP address:
     - Mac: Open Terminal, type: ifconfig
     - Windows: Open CMD, type: ipconfig
     Look for "192.168.X.XXX" or similar

  3. Share with students:
     Instructor: http://YOUR.IP.ADDRESS:3040/instructor.html
     Students: http://YOUR.IP.ADDRESS:3040/student.html

THE FLOW

1. Create session (you)

9. Click "Advance to Phase 3"
